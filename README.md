#### Installing the extension for development

##### Chrome

Browse to [chrome://extensions](chrome://extensions/), tick the **'Developer mode'** option, click **'Load unpacked extension'** and select the `chrome` folder. If all goes well, when you open/restart devtools you should see a **'GA Debugger'** panel.
